#  Productr Frontend

Frontend for **Productr**, a product management dashboard with OTP authentication and secure product operations.

Backend Repo: https://github.com/shreebhagwan03/productr-backend

## Tech Stack
- React
- Axios
- React Router
- Bootstrap

## Features
- OTP Login & Signup
- Protected Routes
- Responsive Dashboard
- Product Create / Edit / Delete
- Image Upload
- Publish / Unpublish Products

## Setup

Create a file named `.env` in the root folder and add:

REACT_APP_API_URL=your_backend_url

Example:

REACT_APP_API_URL=http://localhost:5000/api


Then run:

npm install  
npm start  

## Important
Do NOT upload your `.env` file to GitHub.  
Make sure `.env` is inside `.gitignore`.
